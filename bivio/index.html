biVio Ecosistema Generativo

Este espacio funciona como un laboratorio de experimentación continua. Bajo la co-dirección con Homero Prodan, hemos desarrollado un ecosistema donde la estética y el código convergen para dictar una narrativa visual reactiva.

Cada pieza es un fragmento de un pipeline diseñado para transformar el sonido en datos tangibles, optimizando la carga sensorial para entornos de alta fidelidad digital. No es solo video; es la documentación de un proceso vivo.
